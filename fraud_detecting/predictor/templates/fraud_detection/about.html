About This Project

Insurance Fraud Detection is a modern web application that helps insurance companies and analysts detect potential fraudulent claims using advanced machine learning models. Enter claim details, get instant fraud risk predictions, and analyze your prediction history and statistics.

Features

  • Easy-to-use fraud prediction form
  • Personalized prediction history
  • Analytics dashboard for insights
  • Secure authentication system
  • Modern, responsive UI

About the Author

This project was developed as part of a thesis to demonstrate the application of machine learning in real-world fraud detection scenarios.
